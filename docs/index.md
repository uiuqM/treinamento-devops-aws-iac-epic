## Sobre o Treinamento

**12 encontros · 2h cada · 24h totais**  
Ambiente de laboratório: **AWS Academy Learner Lab**

O treinamento é 100% prático. Cada bloco culmina em um projeto real executado no console AWS:

```
Projeto 1 → Site estático no S3 provisionado com Terraform
Projeto 2 → App Flask em Docker na EC2 com deploy via GitHub Actions
Projeto 3 → API com banco RDS MySQL, stack completa via Terraform
```

Ao final, o aluno sai sabendo provisionar infraestrutura real na AWS usando código — a mesma abordagem adotada por times de engenharia no mercado.

## Regras do Laboratório

1. **Sempre execute `terraform destroy`** ao final de cada sessão
2. **Nunca clique em "Reset"** no Learner Lab — apaga tudo sem recuperação
3. O IP público da EC2 **muda entre sessões** — verificar no início de cada lab
4. Dúvidas durante a semana: canal do curso ou e-mail do instrutor
