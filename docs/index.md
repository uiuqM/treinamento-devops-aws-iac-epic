# Treinamento DevOps AWS IaC

Bem-vindo ao treinamento de **DevOps e Infraestrutura como Código na AWS**!

Este material cobre os principais conceitos e práticas para provisionar e gerenciar infraestrutura na AWS de forma automatizada e escalável.

## O que você vai aprender

- **AWS**: principais serviços para computação, rede, armazenamento e segurança
- **Infraestrutura como Código (IaC)**: Terraform e AWS CDK para provisionar recursos de forma declarativa
- **DevOps**: pipelines de CI/CD, automação e melhores práticas

## Pré-requisitos

- Conta AWS (free tier é suficiente para a maioria dos laboratórios)
- Git instalado localmente
- Conhecimento básico de linha de comando (Linux/macOS/WSL)
